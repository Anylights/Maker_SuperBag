# 《鸭巢行动》- 2D俯视角PVE搜打撤射击游戏 策划案

## 一、游戏概述

**游戏名称**：鸭巢行动（Operation Nest）
**类型**：2D俯视角 PVE 搜打撤射击 + 背包管理 + Roguelite
**核心参考**：逃离鸭科夫、赛菲莉娅、元气骑士

**一句话描述**：在随机生成的危险区域中搜刮物资、击败敌人、安全撤离，用战利品升级基地和打造装备，每一次出击都是一场风险与收益的博弈。

---

## 二、核心循环

```
基地 (安全区): 整理背包 → 升级设施 → 制造/改装装备 → 选择地图
       ↓ 出击
战场 (一局 5~10 分钟): 搜刮 → 战斗 → 收集 → 撤离 (或死亡丢失背包)
       ↓ 撤离成功 / 失败
带出战利品 / 丢失装备(可跑尸捡回)
       ↓
返回基地，继续循环
```

---

## 三、核心玩法系统

### 3.1 搜打撤机制

- **出击准备**: 从仓库选择携带的武器、弹药、药品、背包
- **地图进入**: 每张地图有多个随机出生点
- **搜刮**: 打开箱子/搜索容器需要短暂读条(1~2秒)，期间无法射击
- **战斗**: 击杀敌人掉落物资和装备
- **撤离**: 地图上有2~3个撤离点，到达后倒计时3秒撤离
- **死亡惩罚**: 背包物品掉落在原地，可重新进入地图"跑尸"捡回
- **时间限制**: 每局有5分钟倒计时，超时后环境持续伤害

### 3.2 背包管理系统（格子背包）

- 不同物品占据不同格数，需要合理摆放
- **安全容器**: 死亡后不掉落(初始2×2，可升级到3×3)
- **战术背包**: 死亡后掉落(可搜刮更大的背包替换)
- 物品占格参考: 子弹1×1, 药品1×1, 手枪1×2, 步枪1×3, 护甲2×2

### 3.3 Roguelite 要素

- **随机地图**: 房间模块随机拼接
- **随机敌人**: 种类、位置、巡逻路线随机
- **随机战利品**: 搜刮点物品随机，稀有度不同
- **局内增益**: "芯片"临时被动，仅当局有效
- **永久成长**: 基地升级、蓝图解锁、仓库扩容

### 3.4 武器系统

| 类型 | 特点 | 占格 |
|------|------|------|
| 手枪 | 射速中等，精度高 | 1×2 |
| 冲锋枪 | 射速快，精度低 | 1×2 |
| 步枪 | 伤害高，射速中 | 1×3 |
| 霰弹枪 | 近距离扇形伤害 | 1×3 |
| 近战 | 无弹药消耗 | 1×1 |

### 3.5 敌人设计

| 敌人类型 | 行为模式 | 危险等级 |
|----------|----------|---------|
| 巡逻兵 | 固定路线巡逻，发现玩家后追击 | ★ |
| 哨兵 | 原地驻守，远距离射击 | ★★ |
| 冲锋者 | 发现后快速冲向玩家近战 | ★★ |
| 重装兵 | 移动慢，高血量高伤害 | ★★★ |
| 精英怪 | 特殊技能(投掷手雷/召唤增援) | ★★★★ |
| BOSS | 地图深处，守护高价值物资 | ★★★★★ |

### 3.6 基地系统

| 设施 | 功能 |
|------|------|
| 仓库 | 存储物品，可扩容 |
| 工作台 | 制造弹药/药品 |
| 武器台 | 改装/维修武器 |
| 商店 | 买卖物品 |
| 医疗站 | 恢复生命 |
| 情报室 | 解锁新地图/任务 |

## 四、地图设计

| 地图 | 难度 | 特色 | 主要战利品 |
|------|------|------|-----------|
| 废弃仓库 | ★ | 新手教学，敌人少 | 基础物资 |
| 城市废墟 | ★★ | 复杂建筑结构 | 武器配件 |
| 军事基地 | ★★★ | 高密度敌人，有BOSS | 高级武器 |
| 地下实验室 | ★★★★ | 黑暗环境，视野受限 | 稀有材料 |
| 辐射区 | ★★★★★ | 持续环境伤害 | 顶级装备 |

## 五、开发阶段

### Phase 1 — 核心射击（可玩原型）
1. 玩家移动(WASD) + 瞄准射击(鼠标方向)
2. 1种武器 + 子弹飞行 + 碰撞
3. 3种基础敌人(巡逻、哨兵、冲锋)
4. 1张固定地图 + 战争迷雾
5. HUD(血量/弹药/时间)

### Phase 2 — 搜刮与撤离
6. 可交互容器(搜刮读条)
7. 撤离点机制
8. 简易背包(列表式)
9. 物品掉落系统

### Phase 3 — 背包与基地
10. 格子背包管理
11. 基地场景(仓库+商店)
12. 死亡惩罚 + 跑尸机制

### Phase 4 — Roguelite深度
13. 随机地图生成
14. 局内芯片系统
15. 更多武器/敌人种类
16. BOSS战
